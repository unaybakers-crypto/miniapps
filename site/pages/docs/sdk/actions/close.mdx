# close

Closes the mini app.

![closing the app](/close_preview.png)

<Caption>
  Close the app with `close`.
</Caption>

## Usage

```ts twoslash
import { sdk } from '@farcaster/frame-sdk'

await sdk.actions.close()
```

## Return Value

`void`
